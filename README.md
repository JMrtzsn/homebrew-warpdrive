# homebrew-warpdrive

Homebrew tap for [Warpdrive](https://github.com/JMrtzsn/warpdrive) — Warp terminal with local AI.

## Install

```bash
brew tap johannes-ikea/warpdrive
brew install --cask warpdrive
```

## Update

```bash
brew upgrade --cask warpdrive
```

## Uninstall

```bash
brew uninstall --cask warpdrive
```
